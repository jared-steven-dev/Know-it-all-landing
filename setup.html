Quick Setup Guide

Get Started in Under a Minute

Follow these simple steps to install and configure Know It All Chrome extension. Start using AI-powered features on any webpage instantly.

Requirements

Before you begin, make sure you have the following:

Google Chrome

Version 88 or higher

Gemini API Key

Free from Google AI Studio

Internet Connection

Required for AI features

1

Download the Extension

Get the latest version of Know It All from GitHub.

  • Visit the GitHub repository
  • Click on the green "Code" button
  • Select "Download ZIP" to download the extension files
  • Extract the ZIP file to a location on your computer
üí° Pro Tip: Keep the extracted folder in a permanent location. Don't delete it after installation, as Chrome needs to access these files.
2

Get Your Gemini API Key

You'll need a free API key from Google AI Studio to power the AI features.

  • Go to Google AI Studio
  • Sign in with your Google account
  • Click "Create API Key" button
  • Copy your API key and save it securely
üîí Security Note: Your API key is stored locally in your browser and never shared. Keep it private and don't share it with others.
3

Load the Extension in Chrome

Install the extension in Chrome using Developer Mode.

  • Open Google Chrome and navigate to chrome://extensions/
  • Enable "Developer mode" using the toggle in the top-right corner
  • Click "Load unpacked" button
  • Select the extracted extension folder
  • The extension icon should appear in your Chrome toolbar
‚ö†Ô∏è Important: Make sure to select the folder containing the manifest.json file, not a parent folder.
4

Configure Your API Key

Enter your Gemini API key to activate all AI features.

  • Click on the Know It All extension icon in your Chrome toolbar
  • Click on the "Settings" or gear icon
  • Paste your Gemini API key in the API Key field
  • Click "Save" to store your configuration
‚ú® You're All Set! Your extension is now ready to use. Try selecting any text on a webpage to see the AI features in action.
5

Start Using Know It All

Select any text on any webpage and choose from powerful AI features.

  • Select text on any webpage
  • Choose an action from the popup menu (Explain, Summarize, Translate, etc.)
  • View results instantly in a beautiful side panel
  • Copy or share the AI-generated content

Explain

Get clear explanations of complex text

Summarize

Get concise summaries of long content

Translate

Translate text to any language
