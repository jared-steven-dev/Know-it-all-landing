Frequently Asked Questions

Know It All is an AI-powered Chrome extension that helps you understand anything on the web instantly. Simply select any text on a webpage and choose from six powerful features:
  • Explain: Get clear explanations of complex topics
  • Summarize: Convert lengthy content into concise summaries
  • Translate: Break language barriers with instant translations
  • Quiz Solver: Get answers to quiz questions with explanations
  • Code Generator: Turn problems into working code solutions
  • Quote Generator: Create clever, memorable quotes from any text
The extension uses Google's Gemini AI to provide accurate, context-aware responses in seconds.
Yes! Know It All is completely free to use. You'll need a free Google Gemini API key, which comes with generous usage limits:
  • 15 requests per minute
  • 1,500 requests per day
  • 1 million requests per month
This is more than enough for typical daily use. The API key is free and takes less than 2 minutes to obtain from Google AI Studio.
Getting your API key is quick and easy:
  • Visit Google AI Studio
  • Sign in with your Google account
  • Click "Create API Key"
  • Copy your key and paste it into the extension settings
For detailed step-by-step instructions with screenshots, check out our API Key Setup Guide.
Absolutely! Know It All offers extensive customization options:
  • Language Preferences: Set your preferred language for responses and translations
  • Explanation Types: Choose from 6 styles (causal, sequential, descriptive, comparative, analytical, interpretive)
  • Summary Types: Select from 9 formats (descriptive, evaluative, narrative, executive, and more)
  • Programming Languages: Pick your preferred language for code generation (13+ options)
  • Translation Mode: Use default mode or "Learn Mode" for detailed language breakdowns
Visit our Settings Guide to learn about all customization options.
Yes, your privacy is our priority:
  • Local Storage: Your API key is stored locally in your browser, never on our servers
  • Direct Communication: The extension communicates directly with Google's Gemini API
  • No Data Collection: We don't collect, store, or track your queries or responses
  • Open Source: The extension code is open source and available on GitHub for transparency
Only the text you select is sent to Google's Gemini API for processing, following Google's privacy policies.
Know It All supports 13+ programming languages:
  • Python, JavaScript, TypeScript
  • Java, C++, C#
  • Go, Rust, Ruby
  • PHP, Swift, Kotlin
  • And more!
You can set your preferred language in the settings, or choose "Auto-detect" to let the AI select the best language for each specific problem.
Know It All is currently available as a Chrome extension. It works on:
  • Google Chrome (recommended)
  • Microsoft Edge (Chromium-based)
  • Brave Browser
  • Other Chromium-based browsers
Make sure you're using a recent version of your browser for the best experience.
Installation is quick and easy:
  • Download the extension from the Chrome Web Store or GitHub
  • Click "Add to Chrome" to install
  • Get your free Gemini API key from Google AI Studio
  • Enter the API key in the extension settings
  • Start using all features immediately!
The entire setup takes less than 2 minutes. Check our Quick Setup Guide for detailed instructions.
