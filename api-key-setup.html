API Key Setup Guide

Get Your Free Gemini API Key

Learn how to obtain and configure your Google Gemini API key to unlock all AI-powered features in Know It All extension.

1

Visit Google AI Studio

Google AI Studio is the official platform for accessing Gemini API keys.

  • Open your web browser and navigate to aistudio.google.com/app/apikey
  • You'll be directed to the Google AI Studio API key management page
üí° Quick Access: Click the link above to go directly to the API key creation page. Make sure you're signed in to your Google account.
2

Sign In with Your Google Account

Use any Google account to access AI Studio and create your API key.

  • Click "Sign in" if you're not already logged in
  • Choose your Google account or create a new one
  • Accept the terms of service if prompted
‚ÑπÔ∏è Account Requirements: You can use any personal or work Google account. No special permissions or subscriptions are required for the free tier.
3

Create Your API Key

Generate a new API key that will power the AI features in Know It All.

  • Click the "Create API Key" button
  • Select an existing Google Cloud project or create a new one
  • Wait a few seconds for the key to be generated
  • Your new API key will be displayed on the screen

4

Copy and Save Your API Key

Store your API key securely before configuring it in the extension.

  • Click the copy button next to your API key
  • Save it in a secure location (password manager recommended)
  • You can always return to AI Studio to view or regenerate your key
üí° Pro Tip: Use a password manager like 1Password, LastPass, or Bitwarden to securely store your API key. This makes it easy to access when needed while keeping it safe.
5

Configure the Extension

Add your API key to Know It All to activate all AI features.

  • Click the Know It All extension icon in your Chrome toolbar
  • Click the "Settings" or gear icon in the extension popup
  • Paste your API key into the "API Key" field
  • Click "Save" to store your configuration
  • The extension will verify your key and confirm it's working
